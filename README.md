# 🏢 Freyja — Управління ОСББ

Мобільний Android-застосунок для автоматизації рутинних процесів для мешканців багатоквартирних будинків. Цифровізує документообіг, спрощує фінансовий облік та взаємодію з адміністрацією.

## 🏗 Архітектура
Нативний мобільний застосунок із локальним збереженням даних. Забезпечує високу швидкість відгуку та повну автономність роботи без потреби в інтернет-з'єднанні.

## ⚙️ Основний функціонал
* **💰 Фінансовий моніторинг:** відстеження балансу, заборгованостей та історії оплат.
* **📄 Генерація звітності:** автоматичне формування платіжних квитанцій у форматі PDF.
* **📢 Зворотний зв'язок:** зручна комунікація з адміністрацією через систему оголошень та особистих звернень.
  
## 🛠 Стек технологій
* **Мова розробки:** Kotlin
* **Платформа:** Android SDK
* **База даних:** SQLite
* **Середовище:** Android Studio
